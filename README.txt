SimpleOthello

Javaで作った素朴なAIオセロゲームです。久しぶりに対戦したら負けたので悔しさ晴らすために晒す。

# build
EclipseプロジェクトになっているのでEGitからインポートすればいけるはず。

# to use
最初に出てくるダイアログで先手後手、AIの強さを3段階で設定できます。
正直、レベル3以外は弱すぎるでしょう。ストレス解消にやっつけてやってください。
レベル3は気を抜くと負けるぐらいの微妙な強さです。負けたら猛省です。

＃ Why?
すごく素朴にゼロから何も資料に当たらずに作ったオセロゲームです。
だからいろいろとアラがあるので、Javaを練習したいって人のモルモットとして使えるかなーと思い公開しました。
「アホな書きかたしてるー、ぷふふ」と存分に弄ってください。
forkしてちょっと自分流に直してみるのもおもろいと思いますよ。
